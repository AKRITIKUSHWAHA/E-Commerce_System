## ✨ Features

### 🧑‍💼 Admin Panel
- Full product management — Add, Edit, Delete with multi-image upload (URL or file)
- Stock management — auto-deduction on order, low stock alerts, additive stock update
- Order management — update status (Shipped → Delivered → Cancelled)
- **Date/Month/Year wise order filter**
- Category management — create & toggle visibility on storefront
- Customer management — view all users, ban/unban accounts
- Analytics dashboard — revenue chart (6 months), page views, today's visits
- **Banner management** — add/edit/delete home page banners with gradient presets & live preview
- **Ads management** — Banner, Text bar, Popup ads with position, schedule, color picker & photo upload
- **Contact messages** — view, mark read, reply (saves in DB + sends email), delete
- Auto-refresh dashboard every 30 seconds
- Non-returnable disclaimer toggle per product

### 🛒 Customer Store
- Browse products with filters — category, price range, rating, in-stock
- **Image slider** on product cards with arrows and dot indicators
- Wishlist & Cart with quantity control
- Multi-image product gallery with thumbnails
- Stock count display on each product card (low stock warning ≤5)
- Search products from navbar
- **Quantity selector** before checkout (max = available stock)

### 💳 Checkout & Payments
- Delivery address with **Landmark** & **Alternate phone number** fields
- Razorpay integration — UPI, Cards, Net Banking, Wallets
- HMAC-SHA256 payment signature verification
- Order confirmation page with full details after payment
- Cart items locked in state before clear — success page always shows correctly

### 📦 Order Tracking (User Account)
- Real-time order status: Confirmed → Shipped → Delivered
- Visual step-by-step tracking bar (Flipkart-style)
- Order history with item images & delivery address
- Click product image in order → goes back to product page
- **My Messages tab** — user sees their contact messages + admin replies

### ⭐ Reviews & Ratings
- Star rating system (1–5 stars)
- Review option unlocked **only after order is delivered**
- Average rating displayed on product cards and detail page

### 🖼️ Hero Banners (Configurable)
- Admin can add/edit/delete banners from dashboard
- Gradient background presets (6 colors)
- Image URL support
- Auto-slide every 4 seconds with dot indicators
- Falls back to default banner if none configured

### 📢 Ads System
- Admin creates ads with type: Banner / Text Bar / Popup
- Position: Home Top, Home Middle, Home Bottom, All Pages
- Schedule with start/end dates
- Photo upload or image URL
- Live preview in modal
- Active/Inactive toggle

### 🔐 Authentication
- JWT-based login & registration
- Bcrypt password hashing (10 salt rounds)
- Role-based access control (Admin / Customer)
- Change password from Profile tab
- Protected routes for admin & user pages

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v16+
- [MySQL](https://www.mysql.com/) v8+
- npm

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/steepray-ecommerce.git
cd steepray-ecommerce
```

### 2. Backend Setup

```bash
cd Backend
npm install
npm install nodemailer
```

Create `.env` file (see below), then:

```bash
node server.js
```

Backend runs on: `http://localhost:5000`

### 3. Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

Frontend runs on: `http://localhost:5173`

## 🔐 Environment Variables

Create `.env` inside `Backend/` folder:

```env
# Server
PORT=5000

# Database
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_NAME=ecommerce_db

# JWT
JWT_SECRET=your_jwt_secret_key

# Razorpay
RAZORPAY_KEY_ID=rzp_test_xxxxxxxxxxxxxxxx
RAZORPAY_KEY_SECRET=your_razorpay_secret

# Email (Gmail App Password)
EMAIL_USER=your_gmail@gmail.com
EMAIL_PASS=your_app_password
STORE_NAME=Steepray Store
```

> ⚠️ Do NOT add `=` comments in `.env` — it breaks key parsing.  
> Gmail App Password: Google Account → Security → 2-Step Verification → App Passwords

## 💡 Key Notes

- **Razorpay flow:** Create Order → Popup → HMAC verify → Save order (status = `confirmed`)
- **Stock:** Auto decrements on payment; admin adds stock additively (not replace)
- **Reviews:** Only users with a `delivered` order for that product can submit
- **Banners:** Fetched from DB by Hero component; falls back to default if empty
- **Ads:** `AdsDisplay` component fetches by `position` prop — place anywhere in layout
- **Cart checkout:** Items locked in `useState` before `CLEAR_CART` so success page always works
- **Order filter:** Admin can filter orders by specific Date / Month / Year
- **Contact reply:** Saved in DB + email sent via Nodemailer to user
- **Images:** Stored in `Backend/uploads/` via Multer; served as static files at `/uploads/`
- **Auto refresh:** Home, Admin Dashboard & User Account auto-refresh every 30 seconds silently
- **Tags:** Any unknown tag gets a default gray style; known tags (Casual, Office, Ethnic, etc.) have colored badges
